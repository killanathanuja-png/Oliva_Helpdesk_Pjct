CDD Ticket Flow - Step by Step Guide

From Login to Ticket Closure | Complete walkthrough for CDD users, Clinic Managers, and AOMs

Phase 1: CDD User Raises a Ticket
1

LOGIN

Open http://localhost:8080 in browser

Enter your CDD credentials (email & password)

System redirects to Dashboard

CDD User
2

NAVIGATE TO TICKETS

Click Tickets in the left sidebar

Click View & Update Tickets or All Tickets

You can see all CDD tickets (raised by you and your team)

CDD User
3

CLICK "RAISE TICKET"

Click the + Raise Ticket button (top right, teal color)

A modal/popup opens with the ticket form

Department is auto-set to Clinic

CDD User
4

FILL MANDATORY FIELDS

All fields marked with * are mandatory:

Client Code: Enter client code
Client Name: Enter client name
Description: Describe the complaint
Category: Treatment / Marketing
Sub-Category: Select from dropdown
Service Name: Related service
CRT Name: CRT person name
Primary Doctor: Doctor involved
Therapist Name: Therapist involved
Center: Select clinic center

When you select a Center, the Clinic Manager (CM) and AOM are auto-filled

CDD User
5

SUBMIT TICKET

Click Submit Ticket button

Ticket is created with status: Open

Ticket is auto-assigned to Clinic Manager of the selected center

Ticket code generated: TKT0001, TKT0002, etc.

Email → CDD User (Ticket Created) Email → Clinic Manager (Ticket Assigned) System
Phase 2: Clinic Manager Responds
6

CM RECEIVES TICKET

Clinic Manager logs in → sees the ticket in View & Update Tickets

Ticket shows in their list with status Open

CM clicks on the ticket to open details

Clinic Manager
7

CM WORKS ON TICKET

CM reviews the issue, contacts the client/team

CM can change status to: In Progress

CM can add comments to update the CDD team

CM needs to resolve within 6 working hours (before auto-escalation)

Email → CDD User (Status Update) Clinic Manager
8

CM RESOLVES OR CLOSES

If issue is fixed, CM changes status to:

Resolved or Closed

Adds resolution comment explaining what was done

Email → CDD User (Status Update) Clinic Manager
Auto-Escalation Alert: If CM does NOT resolve within 6 working hours (10AM-6PM IST, Mon-Sat), the system automatically escalates to AOM (L1). Skip to Step 10.
Phase 3: CDD Reviews the Resolution
9

CDD REVIEWS RESOLVED TICKET

CDD user opens the resolved/closed ticket

Reviews if the issue is truly resolved

Two options:

All Good → Final Close

Click Final Close

Status: Final Closed

TICKET COMPLETE

Not Resolved → Reopen

Click Reopen

Status: Reopened by CDD

Goes back to CM/Assignee

Email → Assignee (Status Update) CDD User
Phase 4: Auto-Escalation (If CM Doesn't Resolve)
10

AUTO-ESCALATION TO L1 (AOM) — After 6 Working Hours

System automatically escalates if CM hasn't resolved

Status changes to: Escalated to L1

Ticket reassigned to the center's AOM (Area Operations Manager)

System adds comment: "Auto-escalated to L1 AOM. Originally assigned to [CM name]"

Email → AOM (Ticket Assigned) System (Automatic)
11

AOM WORKS ON TICKET

AOM logs in → sees escalated ticket in their list

Ticket shows: Assigned To: [AOM Name] (L1 AOM)

AOM can only: Resolve or Close

AOM adds resolution comment

Email → CDD User (Status Update) AOM
Second Escalation: If AOM also doesn't resolve within 14 total working hours (6+8), the system auto-escalates to L2 Manager. Same process — L2 resolves and closes.
12

AUTO-ESCALATION TO L2 — After 14 Working Hours

Status: Escalated to L2 | Assigned to L2 Manager

L2 Manager resolves → CDD does Final Review (Step 9)

Email → L2 Manager (Ticket Assigned) System (Automatic)
Manual Actions (CDD Can Also Do These Anytime)
M1

MANUAL ACKNOWLEDGE

CDD clicks Acknowledge to confirm ticket receipt

Status: Acknowledged

CDD User
M2

MANUAL ESCALATE L1

CDD can manually escalate to AOM without waiting for auto-escalation

Status: Escalated to L1

CDD User
M3

MANUAL ESCALATE L2

CDD can manually escalate to L2 Manager

Status: Escalated to L2

CDD User
Quick Reference: Complete Timeline
1. Login → 2. Go to Tickets → 3. Click Raise Ticket → 4. Fill Fields → 5. Submit → 6. CM Gets Ticket → 7. CM Works → 8. CM Resolves → 9. CDD Reviews → Final Close
If CM doesn't resolve: → 10. Auto-Escalate AOM (6hrs) → 11. AOM Resolves → CDD Reviews → Final Close
If AOM doesn't resolve: → 12. Auto-Escalate L2 (14hrs) → L2 Resolves → CDD Reviews → Final Close
